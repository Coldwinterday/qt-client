```cpp

m_updateTimer = new QTimer(this);
connect(m_updateTimer, &QTimer::timeout, this, &BarChartWidget::updateData);
```

**数据更新逻辑**：
```cpp
void updateData() {
    // 随机增减每个柱子的值（±10）
    for (auto &data : m_data) {
        int delta = (rand() % 21) - 10;  // -10 到 +10
        data.value = qBound(50, data.value + delta, 350);
    }
    update();  // 触发重绘
}
```

## 四、技术要点

### 4.1 星标评分显示
```cpp
QString getStarString(double rating) {
    int fullStars = (int)rating;
    bool hasHalf = (rating - fullStars) >= 0.5;
    
    QString stars;
    for (int i = 0; i < fullStars; ++i) {
        stars += "⭐";
    }
    if (hasHalf) {
        stars += "⭐";  // 或使用半星符号
    }
    return stars + QString(" %1分").arg(rating);
}
```

### 4.2 批量更新对话框
