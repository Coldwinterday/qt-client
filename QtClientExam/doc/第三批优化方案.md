## 二、详细实现方案

### 2.1 界面切换优化方案

#### 2.1.1 问题分析
当前QStackedWidget切换是瞬间完成，没有过渡效果。

#### 2.1.2 解决方案
使用QPropertyAnimation实现淡入淡出效果：

```cpp
// MainWindow.h 添加
private:
    void switchPageWithAnimation(int index);
    QGraphicsOpacityEffect *m_opacityEffect;

// MainWindow.cpp 实现
void MainWindow::switchPageWithAnimation(int index) {
    QWidget *currentWidget = m_stackedWidget->currentWidget();
    QWidget *nextWidget = m_stackedWidget->widget(index);
    
    // 淡出当前页面
    QPropertyAnimation *fadeOut = new QPropertyAnimation(m_opacityEffect, "opacity");
    fadeOut->setDuration(200);
    fadeOut->setStartValue(1.0);
    fadeOut->setEndValue(0.0);
    
    connect(fadeOut, &QPropertyAnimation::finished, [this, index, fadeOut]() {
        m_stackedWidget->setCurrentIndex(index);
        
        // 淡入新页面
        QPropertyAnimation *fadeIn = new QPropertyAnimation(m_opacityEffect, "opacity");
        fadeIn->setDuration(200);
        fadeIn->setStartValue(0.0);
        fadeIn->setEndValue(1.0);
        fadeIn->start(QAbstractAnimation::DeleteWhenStopped);
        
        fadeOut->deleteLater();
    });
    
    fadeOut->start();
}
```

### 2.3 Tab文字横向排布方案

#### 2.3.1 问题分析
当前QTabWidget设置为West位置，默认文字是竖向排列。

#### 2.3.2 解决方案A：使用样式表（推荐）
```cpp
m_tabWidget->setStyleSheet(
    "QTabBar::tab { "
    "   writing-mode: horizontal-tb; "  // 横向文字
    "   padding: 10px 20px; "
    "   min-height: 40px; "
    "}"
);
```

#### 2.3.3 解决方案B：自定义TabBar
```cpp
class HorizontalTabBar : public QTabBar {
protected:
    QSize tabSizeHint(int index) const override {
        QSize size = QTabBar::tabSizeHint(index);
        // 交换宽高，使文字横向显示
        return QSize(size.height(), size.width());
    }
    
    void paintEvent(QPaintEvent *event) override {
        QPainter painter(this);
        // 自定义绘制横向文字
    }
};
```

#### 2.3.4 解决方案C：改用左侧按钮组（最简单）
```cpp
// 不使用QTabWidget，改用QVBoxLayout + QPushButton
QVBoxLayout *navLayout = new QVBoxLayout();
QStringList tabs = {"轮播图", "柱状图", "面板", "列表", "表格"};

for (int i = 0; i < tabs.size(); ++i) {
    QPushButton *btn = new QPushButton(tabs[i]);
    btn->setCheckable(true);
    connect(btn, &QPushButton::clicked, [this, i]() {
        switchPageWithAnimation(i);
    });
    navLayout->addWidget(btn);
}
```

**推荐方案C**：最灵活，样式控制最好。

### 2.4 修复用户数据添加崩溃方案

#### 2.4.1 崩溃原因排查
可能原因：
1. QTableWidgetItem未正确初始化
2. 访问了空指针
3. 信号槽连接问题
4. 排序功能与插入冲突

#### 2.4.2 解决方案
```cpp
void DataTableWidget::onAddDataClicked() {
    // 输入数据...
    
    // 关键：添加前禁用排序
    m_tableWidget->setSortingEnabled(false);
    
    int row = m_tableWidget->rowCount();
    m_tableWidget->insertRow(row);
    
    // 确保每个item都正确创建
    QTableWidgetItem *idItem = new QTableWidgetItem(QString::number(getNextId()));
    QTableWidgetItem *nameItem = new QTableWidgetItem(name);
    QTableWidgetItem *ageItem = new QTableWidgetItem(QString::number(age));
    QTableWidgetItem *emailItem = new QTableWidgetItem(email);
    QTableWidgetItem *statusItem = new QTableWidgetItem("在线");
    
    // 设置对齐方式
    idItem->setTextAlignment(Qt::AlignCenter);
    ageItem->setTextAlignment(Qt::AlignCenter);
    statusItem->setTextAlignment(Qt::AlignCenter);
    
    // 设置颜色
    statusItem->setForeground(QBrush(QColor(76, 175, 80)));
    
    // 插入item
    m_tableWidget->setItem(row, 0, idItem);
    m_tableWidget->setItem(row, 1, nameItem);
    m_tableWidget->setItem(row, 2, ageItem);
    m_tableWidget->setItem(row, 3, emailItem);
    m_tableWidget->setItem(row, 4, statusItem);
    
    // 重新启用排序
    m_tableWidget->setSortingEnabled(true);
    
    QMessageBox::information(this, "成功", "数据添加成功！");
}
```

#### 2.5.2 实现方案
```cpp
// CarouselWidget.h 添加
private slots:
    void onUploadClicked();

private:
    QPushButton *m_uploadBtn;

// CarouselWidget.cpp 实现
void CarouselWidget::onUploadClicked() {
    // 打开文件对话框
    QString filePath = QFileDialog::getOpenFileName(
        this,
        "选择图片",
        QDir::homePath(),
        "图片文件 (*.png *.jpg *.jpeg *.bmp)"
    );
    
    if (filePath.isEmpty()) {
        return;
    }
    
    // 验证图片
    QPixmap pixmap(filePath);
    if (pixmap.isNull()) {
        QMessageBox::warning(this, "错误", "无效的图片文件！");
        return;
    }
    
    // 添加到轮播列表
    QString title = QString("自定义图片 %1").arg(m_items.size() + 1);
    CarouselItem newItem(filePath, title);
    m_items.append(newItem);
    
    // 更新指示器
    QLabel *indicator = new QLabel(this);
    indicator->setFixedSize(12, 12);
    indicator->setStyleSheet("QLabel { background-color: #bdbdbd; border-radius: 6px; }");
    m_indicators.append(indicator);
    m_indicatorLayout->insertWidget(m_indicatorLayout->count() - 1, indicator);
    
    // 切换到新图片
    switchToImage(m_items.size() - 1);
    
    QMessageBox::information(this, "成功", "图片上传成功！");
}
```
