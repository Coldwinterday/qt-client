# QtClientExam 项目记忆宫殿

## 一、项目概览

**项目名称**：QtClientExam - Qt客户端考核项目  
**技术栈**：Qt 5.14.2 + C++11 + MinGW64  
**核心目标**：实现登录、启动页、主界面（轮播图/柱状图/面板/列表/表格）+ 窗口管理

---

## 二、架构设计

### 2.1 整体架构（MVC + 组件化）

```
main.cpp
    ↓
LoginWindow (硬编码校验)
    ↓ 成功
SplashWindow (启动进度0→100%)
    ↓ 完成
MainWindow (主容器)
    ├── CarouselWidget (轮播图)
    ├── BarChartWidget (柱状图)
    ├── PanelWidget (面板网格)
    ├── AppListWidget (应用列表)
    └── DataTableWidget (数据表格)
```

### 2.2 设计模式应用

| 模式 | 应用位置 | 作用 |
|------|---------|------|
| 单例模式 | AppConfig | 全局配置管理（主题色、动画时长） |
| 工厂模式 | DataFactory | 统一生成假数据（轮播/列表/表格） |
| 观察者模式 | 信号槽 | 窗口跳转、数据更新通知 |

### 3.2 SplashWindow（启动页）

**职责**：显示启动进度动画（0→100%）  
**关键代码位置**：`Sources/SplashWindow.cpp:startProgress()`

**核心技术**：
- `QTimer` 定时更新进度
- `QProgressBar` 显示百分比
- `QPropertyAnimation` 平滑过渡

**动画实现**：
```cpp
QTimer *timer = new QTimer(this);
connect(timer, &QTimer::timeout, [this]() {
    currentProgress += 2;
    progressBar->setValue(currentProgress);
    if (currentProgress >= 100) {
        timer->stop();
        emit loadComplete();  // 跳转MainWindow
    }
});
timer->start(50);  // 每50ms更新一次
```

### 3.5 BarChartWidget（柱状图）

**职责**：QPainter绘制动态柱状图，支持渐变  
**关键代码位置**：`Sources/BarChartWidget.cpp:paintEvent()`

**核心技术**：
- 重写 `paintEvent()`
- `QPainter` 绘制矩形
- `QLinearGradient` 渐变填充
- `QTimer` 模拟数据更新

**绘制逻辑**：
```cpp
void BarChartWidget::paintEvent(QPaintEvent *event) {
    QPainter painter(this);
    painter.setRenderHint(QPainter::Antialiasing);
    
    int barWidth = width() / (values.size() * 2);
    for (int i = 0; i < values.size(); ++i) {
        int barHeight = (values[i] / 100.0) * height();
        
        // 渐变色
        QLinearGradient gradient(0, height() - barHeight, 0, height());
        gradient.setColorAt(0, QColor(100, 200, 255));
        gradient.setColorAt(1, QColor(50, 100, 200));
        
        painter.fillRect(i * barWidth * 2, height() - barHeight, 
                        barWidth, barHeight, gradient);
    }
}
```

## 四、关键技术点

### 4.1 信号槽机制

**窗口跳转链路**：
```cpp
// LoginWindow
connect(loginBtn, &QPushButton::clicked, this, &LoginWindow::onLoginClicked);
emit loginSuccess();  // 发送信号

// main.cpp
connect(loginWindow, &LoginWindow::loginSuccess, [&]() {
    loginWindow->hide();
    splashWindow->show();
    splashWindow->startProgress();
});
```

### 4.2 动画系统

**QPropertyAnimation 使用模式**：
```cpp
QPropertyAnimation *anim = new QPropertyAnimation(target, "propertyName");
anim->setDuration(500);           // 持续时间
anim->setStartValue(startValue);  // 起始值
anim->setEndValue(endValue);      // 结束值
anim->setEasingCurve(QEasingCurve::InOutQuad);  // 缓动曲线
anim->start(QAbstractAnimation::DeleteWhenStopped);
```

### 4.3 自定义绘制

**QPainter 绘制流程**：
```cpp
void CustomWidget::paintEvent(QPaintEvent *event) {
    QPainter painter(this);
    painter.setRenderHint(QPainter::Antialiasing);  // 抗锯齿
    
    // 设置画笔
    QPen pen(Qt::black, 2);
    painter.setPen(pen);
    
    // 设置画刷（填充）
    QBrush brush(Qt::blue);
    painter.setBrush(brush);
    
    // 绘制图形
    painter.drawRect(x, y, width, height);
}
```

### 4.4 资源管理

**QRC 资源系统**：

```xml
        <file>images/slide2.jpg</file>
    </qresource>

```

**代码中使用**：

## 六、常见问题

### Q1: 编译报错找不到Qt头文件？
**A**: 检查 .pro 文件中 `QT += core gui widgets` 是否正确配置。

### Q2: 动画不流畅，有卡顿？
**A**: 
1. 检查是否在主线程执行耗时操作
2. 使用 `QPropertyAnimation` 而非手动循环更新
3. 启用抗锯齿：`painter.setRenderHint(QPainter::Antialiasing)`

### Q3: 轮播图切换时图片闪烁？
**A**: 使用 `QGraphicsOpacityEffect` 实现淡入淡出，避免直接切换。

### Q4: 表格数据量大时滚动卡顿？
**A**: 
1. 使用 `QTableView` + 自定义Model（而非QTableWidget）
2. 实现懒加载，只渲染可见行
3. 禁用不必要的特效
